# **App Name**: SaaS Genesis

## Core Features:

- Course Content Delivery: Provides access to a structured series of video lessons that teach the basics of SaaS, MVP, and Micro SaaS development using Vibe Coding and modern tools.
- Hands-on Project: Guides the student through the development of a functional Micro SaaS, covering interface design, database integration, and version control.
- Code Assistance Tool: An AI-powered tool assists in Vibe Coding tasks, helping with code generation and debugging based on best practices and the course curriculum. The LLM will use reasoning to decide if there are relevant suggestions from best practices to make to the user's code.
- Deployment Guidance: Offers step-by-step instructions for deploying the Micro SaaS using Firebase, covering environment setup, production deployment, and common pitfalls.
- Monetization Strategies: Explains various monetization models suitable for Micro SaaS, including subscription, freemium, one-time access, and B2B, helping users validate and monetize their projects.
- GitHub Integration: Integrates with GitHub for version control, allowing users to manage their project's code, collaborate with others, and ensure project backup and organization.
- FAQ and Support: Provides a FAQ section addressing common questions about the course and offers support for resolving technical and conceptual issues.

## Style Guidelines:

- Primary color: Soft purple (#A091EA) for a premium, modern look, hinting at the combination of tech and creativity.
- Background color: Dark gray (#282A3A) for a dark mode aesthetic, creating a sleek and immersive experience.
- Accent color: Light lavender (#D7D1F3) to highlight key elements and CTAs, ensuring they stand out against the dark background.
- Headline font: 'Space Grotesk' (sans-serif) for a bold, contemporary feel.
- Body font: 'Inter' (sans-serif) for clear, readable text. Font Pairing: 'Space Grotesk' for headlines, 'Inter' for body.
- Use modern, minimalist icons to represent course modules, features, and benefits, enhancing visual appeal and understanding.
- Mobile-first, fully responsive layout with clear sections, short blocks of text, and strategically placed CTAs to maximize scannability and conversions.
- Subtle glow effects, smooth transitions, and hover animations to enhance user engagement and provide a premium feel.
